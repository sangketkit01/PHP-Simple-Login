Cookie : Information about a user stored in a user's web-browser targeted advertisement , browsing preference and other non-sensitive data

Session : Super global variable used to store information on a user to be used across multiple pages. A user is assigned a session-id
          ex.login credentials

Server : Super global variable that contains headers , paths and scripts locations. The entries in the array are created by the web server.
         Show nearly everything you need to know about the current web page environment.

hashing : Transforming sensitive data (password) into letters , numbers and/or symbols via a mathematics process. (similar to encryption).
           Hides the original data from 3rd parties.
